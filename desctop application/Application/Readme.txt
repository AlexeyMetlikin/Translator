Приложение "Translator". 
Выполняет перевод исходного текста на указанный язык с помощью API переводчика от компании Яндекс (https://tech.yandex.ru/translate/).
Над текстовым полем исходного текста выбираем язык, с которого выполняем перевод.
Над текстовым полем для перевода выбираем язык, на который будем переводить текст.

Для работоспособности приложения перед запуском необходимо заполнить актуальный ключ в файле settings.ini
Актуальный ключ можно получить на сайте https://tech.yandex.ru/translate/

Формат файла settings.ini: 
1. На первой строке указывается наименование секции в формате ([api-key]), не изменяется
2. На второй строке 'key=Актуальный_ключ'
