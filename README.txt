# Translator
Translator application based on yandex API
exe-файл приложения расположен в каталоге Application/Translator.exe
В каталоге TranslatorSolution расположен проект приложения Translator для VS2015, а также проект Translator.Tests c unit-тестами

Приложение выполняет перевод исходного текста на указанный язык с помощью API переводчика от компании Яндекс (https://tech.yandex.ru/translate/).
Над текстовым полем исходного текста выбираем язык, с которого выполняем перевод.
Над текстовым полем для перевода выбираем язык, на который будем переводить текст.

Для работоспособности приложения перед запуском необходимо заполнить актуальный ключ в файле settings.ini
Актуальный ключ можно получить на сайте https://tech.yandex.ru/translate/
